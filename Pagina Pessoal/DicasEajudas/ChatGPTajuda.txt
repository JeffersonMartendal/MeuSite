Claro, vou explicar o código que você compartilhou em termos simples para facilitar a compreensão.

Este código é escrito em JavaScript e jQuery e é usado para criar um efeito de rolagem suave quando você clica em links de navegação em um site. Aqui está o que cada parte do código faz:

$(document).ready(function(){ ... });:
Isso é uma função que espera até que o documento HTML esteja totalmente carregado antes de executar qualquer código JavaScript. Isso é importante para garantir que o código funcione corretamente após a página ser carregada.

$("nav a").on('click', function(){ ... });:
Isso seleciona todos os elementos de âncora <a> dentro de um elemento <nav> (normalmente usado para criar menus de navegação) e atribui uma função a ser executada quando qualquer um desses links for clicado.

var link = $(this).attr('href');:
Aqui, o código está pegando o valor do atributo href do link que foi clicado e armazenando-o na variável link. O $(this) se refere ao link atualmente clicado.

$('html,body').animate({scrollTop: ($(link).offset().top - 100)}, 'slow');:
Esta linha é onde a animação de rolagem suave acontece. Ela faz o seguinte:

$('html,body'): Isso seleciona o elemento HTML e o elemento do corpo da página, que são os elementos que você deseja rolar suavemente.
.animate({scrollTop: ($(link).offset().top - 100)}, 'slow'): Isso inicia uma animação que faz a página rolar suavemente até a posição vertical especificada. scrollTop define a posição vertical em que a página será rolada.
$(link).offset().top pega a posição vertical do elemento relacionado ao link clicado.
- 100 é subtraído da posição para dar um pequeno espaço (100 pixels) acima do elemento de destino após a rolagem.
'slow' é a velocidade da animação. Neste caso, 'slow' indica que a rolagem será feita lentamente para dar um efeito suave.
return false;:
Isso impede o comportamento padrão do link, que seria a navegação para outra página. Em vez disso, a função faz a rolagem suave, mas a página não muda de URL.

Em resumo, esse código jQuery faz com que os links de navegação dentro de um elemento <nav> rolem suavemente para as seções correspondentes da página quando são clicados, em vez de simplesmente saltar para essas seções. Isso cria uma experiência de usuário mais agradável ao navegar em um site.
